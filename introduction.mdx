## What is Cursor?

Cursor is an AI-first code editor built to make you extraordinarily productive. It combines the power of artificial intelligence with a familiar, intuitive interface. Key features include:

- Tab-based code prediction that anticipates your next edit
- Deep codebase understanding for intelligent answers and references
- Natural language code editing
- Built-in MCP (Model Control Protocol) server for enhanced AI capabilities
- Seamless integration with your existing development tools and extensions
