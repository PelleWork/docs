## Color Usage Guidelines

### Text Colors
- On light backgrounds: Black text with Mint Green - Dark for headlines
- On dark backgrounds: White text with Mint Green - Light for headlines

### Background Implementation
- Use Petroleum Green for dark mode backgrounds
- Use Sea Green for light mode backgrounds
- Create visual interest by varying background colors
- Implement background gradients for full-page backgrounds

### Accessibility
All color combinations have been carefully selected to meet WCAG (Web Content Accessibility Guidelines) requirements, ensuring optimal contrast and readability across all digital media.
