# Downloads & Information

## Getting Cursor

Cursor is available for all major platforms. Here's how to get started:

### System Requirements
- Windows 10/11, macOS 10.15+, or Linux
- 4GB RAM minimum (8GB recommended)
- 2GB free disk space

### Download Links
- [Windows Download](https://cursor.sh/download/windows)
- [macOS Download](https://cursor.sh/download/mac)
- [Linux Download](https://cursor.sh/download/linux)

## Key Features

### AI-Powered Development
- Intelligent code completion
- Natural language to code conversion
- Context-aware suggestions
- Built-in MCP server for enhanced AI capabilities

### Modern Development Experience
- Fast and responsive interface
- Built-in terminal
- Git integration
- Extensions support
- Multi-cursor editing

### Collaboration Features
- Real-time collaboration
- Code sharing
- Team workspaces
- Comment and review system

## Quick Start Guide

1. Download and install Cursor for your platform
2. Launch Cursor and sign in with your account
3. Start a new project or open an existing one
4. Begin coding with AI assistance
